# 🌦️ WeatherLive – Real-Time Weather Web App

WeatherLive is a modern and responsive weather web application that provides real-time weather updates, hourly forecasts, and a 5-day forecast for any city.  
The project features a clean **glassmorphism UI**, smooth interactions, and dynamic data powered by a weather API.

This project is suitable for **portfolio showcase**, **internship tasks**, and **front-end + backend practice**.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 📍 Detect current location weather  
- 🌡️ Real-time temperature & feels-like data  
- ⏱️ Hourly weather forecast  
- 📅 5-day weather forecast  
- 🌬️ Wind speed, humidity, pressure & visibility  
- 🌅 Sunrise & sunset timings  
- 🎨 Modern glassmorphism UI with gradients  
- 📱 Fully responsive design

⚙️ Installation & Setup
Step 1: Clone the repository
```text
git clone https://github.com/your-username/weatherlive.git
```
Step 2: Navigate to the project folder
```text
cd weatherlive
```
Step 3: Install dependencies
```text
npm install
```
Step 4: Create .env file
```text
API_KEY=your_openweathermap_api_key
PORT=3000
```
Step 5: Start the server
```text
node server.js
```
Step 6: Open in browser
```text
http://localhost:3000
```
